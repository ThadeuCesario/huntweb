<strong>API</strong>
<p>
Como buscamos dados de uma API Rest? 
</p>
<span>
Vamos utilizar uma biblioteca chamada <strong>axios</strong>, para conseguirmos obter dados
de uma determinada API.
Para instalar essa biblioteca basta executar o comando:
</span>
<br/>
<code>
yarn add axios
</code>
